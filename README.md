# MP_datawrangling

This is Rae's repo for the Jasper Slingsby's data management honours module. 
In this repo you will find data and code concerning trait-based analysis of microplastic consumption in marine invertebrates in the Galpin salt marsh, South Africa (data supplied by William Froneman) 
data folder contains the raw data 
code folder contains the R code 
README.qmd is a quarto file that contains further information about the study, including species used and methods. It also contains the R code laid out nicely with the figure at the bottom. 
There is a binder badge on this README as well.
